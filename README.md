O **LAB028** é um projeto de rádio web focado no conceito *Open Format*, unindo a estética urbana do funk e trap com a sofisticação da música eletrônica. Desenvolvido para ser o epicentro sonoro do **DJ JOTTAPE 028**, o site oferece uma experiência imersiva com visual *Glassmorphism* e alta tecnologia de transmissão.

---

## ⚡ Funcionalidades

*   **Player em Tempo Real:** Conexão direta com servidores de streaming (Zeno.fm, AzuraCast, etc.).
*   **SPA (Single Page Application):** Navegação fluida entre páginas (Home, Notícias, Sobre, Chat) sem recarregar o navegador.
*   **Experiência Imersiva:** Cursor personalizado, fundo animado com gradientes dinâmicos e iluminação ambiente via CSS.
*   **Chat Interativo:** Espaço para interação da comunidade durante as lives.
*   **Design Responsivo:** Totalmente adaptado para dispositivos móveis e desktops.

## 🚀 Como Executar o Projeto

1.  Faça o download do arquivo `index.html`.
2.  Abra o arquivo no seu editor de código (VS Code, por exemplo).
3.  No arquivo `index.html`, procure pela linha:
    ```javascript
    const audio = new Audio("COLOQUE_AQUI_O_LINK_DA_SUA_TRANSMISSAO");
    ```
4.  Substitua o texto pelo link direto da sua transmissão de áudio.
5.  Abra o arquivo em qualquer navegador moderno.

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estrutura semântica e acessível.
*   **CSS3 Avançado:** Animações de keyframes, Glassmorphism e filtros de desfoque.
*   **JavaScript (Vanilla):** Lógica de navegação SPA, manipulação de áudio e efeitos de cursor.
*   **Google Fonts:** Utilização das fontes `Syncopate` e `Outfit`.
